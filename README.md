# MetaTrader 5

MetaTrader 5 plugin for bt_api plugin for `bt_api`.

## Features

- MT5 trading interface\n- Multi-asset support

## Installation

```bash
pip install bt_api_mt5
```

## Usage

```python
from bt_api_bt_api_mt5 import mt5
# Use the exchange
```

## Architecture

The package follows the standard bt_api plugin structure with feeds, containers, and gateway adapters.
